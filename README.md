# Pure CSS Power — 0 Lines of JavaScript

This website is designed to demonstrate the power of modern CSS and HTML. It has been built without a single line of JS. Every interaction, animation, layout trick, and UI component on the page is built with HTML and CSS alone.

**[Live Demo →](index.html)**

---

## What this is

Features that once required JavaScript libraries — scroll animations, tooltips, accordions, tabs, parent-aware styling, adaptive layouts — are now native HTML & CSS. This page shows them off in one place, as live demos alongside the actual code that drives them and key documentation that supports them.

It's equal parts showcase and reference. Its existence proves just how powerful native HTML and CSS has become, and acts as a proof of concept for modern web development.

---

## Running it

No build step. No npm. Open `index.html` in a browser.

```bash
open index.html
# or
npx serve .
```

For the best experience use Chrome 125+ or Safari 17.4+. Some features (Anchor Positioning, View Transitions) have partial support in older browsers — the page degrades gracefully.

---

## Philosophy

> The browser is your framework. It always has been.

The goal is not to avoid all JavaScript on principle — it's to show that a large percentage of problems that were previously handed to JS (and the libraries built around them) can now be solved by the browser. Less script means faster parse times, fewer dependencies, and behaviour that degrades gracefully when scripts fail to load.
